Edit Campaign/Program

The Ad Campaign is a collection of several Ads in a single theme. For example, you can create a "Year End Promo" Ad Campaign that contain services or goods you want to advertise like clothes promo Ad, shoes promo Ad, and so on.

 
 
 
 


All fields needs to be filled, except for the Description field.
